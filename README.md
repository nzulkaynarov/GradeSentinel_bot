# 🛡️ GradeSentinel

**GradeSentinel** — это автоматизированная система мониторинга школьной успеваемости. Бот отслеживает изменения в Google Таблицах (электронных дневниках) и мгновенно уведомляет родителей в Telegram о новых оценках.



## ✨ Основные возможности
* 👨‍👩‍👧‍👦 **Семейные аккаунты:** Поддержка связи «многие-ко-многим» (несколько родителей — несколько детей).
* 📱 **Авторизация по контакту:** Безопасный вход через верификацию номера телефона.
* ⚡ **Мгновенные уведомления:** Опрос таблиц каждые 5 минут.
* 📊 **История оценок:** Просмотр архива оценок прямо в чате бота.
* 🐳 **Docker Native:** Легкое развертывание на Raspberry Pi или любом VPS.
