# 🛡️ GradeSentinel
[v2.0 Docker-based]

**GradeSentinel** — это мощная и умная система мониторинга школьной успеваемости. Бот непрерывно отслеживает изменения в Google Таблицах (электронных дневниках) и мгновенно уведомляет родителей в Telegram о новых оценках их детей.

Система построена на архитектуре "Один ко Многим" и "Многие ко Многим", позволяя легко управлять сложными семейными связями, различными ролями и масштабами.

---

## ✨ Основные возможности

### � Мульти-ролевая Семейная Система
* **Гибкие связи:** Один родитель может состоять в нескольких разных семьях. У одной семьи может быть несколько детей.
* **Иерархия доступа:** 
  * 👑 **Супер-Администратор**: Полный контроль над системой, создание семей, привязка пользователей, глобальная статистика и массовые рассылки.
  * 👨‍👩‍👦 **Глава семьи (Head)**: Управление своей семьей, добавление/удаление старших родственников (бабушек, дедушек), просмотр оценок.
  * 👵 **Старший родственник (Senior/Parent)**: Режим "Только чтение" — получение уведомлений об оценках и запрос истории.

### �️ Безопасность и Валидация
* **Авторизация по контакту:** Никто посторонний не сможет зайти в бота. Доступ выдается только людям с утвержденным номером телефона (формат `998XXXXXXXXX`).
* **Изолированный UI:** Бот показывает только те кнопки и меню, которые соответствуют текущей роли пользователя (Senior не видит кнопок Главы, а Глава не видит админ-панель).

### ⚙️ Мониторинг и Оповещения
* ⚡ **Snapshot Engine:** Бот делает "снимки" дневников и находит точную разницу (конкретную новую оценку), чтобы не присылать мусор.
* 🕒 **Интервалы:** Опрос Google API происходит каждые 5 минут, с защитой от блокировок `Exponential Backoff`.
* � **История на лету:** Быстрый запрос последних оценок ребенка прямо в чате через удобные Inline-кнопки.

### 💬 Обратная связь и Community
* **Direct Feedback:** Пользователи могут отправить сообщение/фото в службу поддержки через кнопку `💬 Поддержка` прямо из бота в закрытую группу администраторов.
* **Replies:** Администраторы могут отвечать на вопросы пользователей прямо из группы через функцию "Ответить" (Reply).
* **Broadcast:** Супер-Админ может отправлять глобальные новости (текст + медиа) всем зарегистрированным пользователям одной кнопкой `📢 Рассылка`.

## 🚀 Быстрый старт (Deploy)

Бот полностью упакован в `Docker`, что делает деплой на Raspberry Pi или VPS максимально простым.

### Требования
* Docker и Docker Compose
* Ключ `credentials.json` (Google Service Account) формата Web/Service.
* Токен Telegram бота от `@BotFather`

### Установка (3 шага)
1. Склонируйте конфигурацию и настройте переменные среды:
   ```bash
   cp .env.example .env
   # Отредактируйте .env: добавьте BOT_TOKEN, ADMIN_ID и ADMIN_GROUP_ID
   ```
2. Положите ключ Google API (переименуйте в `credentials.json`) в папку `config/`.
3. Запустите магию:
   ```bash
   docker-compose up -d --build
   ```

*(Для ознакомления с внутренней архитектурой БД смотрите `Docs/Project_overview.md`)*
